== Description ==
Công cụ chia sẻ tự động - Công cụ hỗ trợ chia sẻ bài viết lên mạng xã hội và các diễn đàn một cách nhanh chống .
Giúp tăng sự tương tác vối website qua mạng xã hội và forum

== Installation ==

1. Mở menu plugins trong trang admin wordpress và tìm kiếm plugin với từ khóa Công cụ hút khách. Nhấn vào cài đặt.
2. Kích hoạt plugin. Bạn sẽ thấy được menu **Chia sẻ tự động** bên trái menu trang quản lý của bạn.
3. Hệ thống sẽ tự cung cấp cho bạn một tài khoản miễn phí để sử dụng công cụ hút khách.
4. Hệ thống sẽ tự động chuyển bạn về trang quản lý công cụ, bạn có thể thêm mới và chỉnh sửa công cụ.
